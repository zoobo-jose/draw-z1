# DEMARRAGE DU PROJET

    PREREQUIS :
        - NODE js
        - Une connexion internet pour acceder a la base de donnees et installer les dependances du projet

    COMMANDES:
        Depuis la racine du projet , ouvrir une invite de commande et entrez les 3 commandes suivante:
            - cd server
            - npm install 
            - npm run serve
        Enfin ouvrir le fichier index.html dans un navigateur de votre choix (firefox,chrome ,edge , ...)
